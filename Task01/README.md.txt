# Описание файлов данных

## ratings.csv
Файл содержит рейтинги фильмов от пользователей.
Структура:
- userId - идентификатор пользователя;
- movieId - идентификатор фильма;
- rating - оценка пользователя;
- timestamp - временная метка выставления оценки.

## movies.csv
Файл содержит информацию о фильмах.
Структура:
- movieId - уникальный идентификатор фильма;
- title - название фильма (включает год выпуска в скобках);
- genres - перечень жанров.

## tags.csv
Файл содержит набор данных о тегах (метках), которые пользователи присваивают фильмам.
Структура:
- userId - yникальный идентификатор пользователя, который присвоил тег ;
- movieId - уникальный идентификатор фильма, к которому относится тег;
- tag - текст тега, который пользователь ассоциирует с фильмом;
- timestamp (целое число) - временная метка, указывающая, когда пользователь присвоил этот тег фильму.

## genres.txt
Назначение: этот файл содержит список всех возможных жанров фильмов, которые используются в системе.
Формат: простой текстовый файл, где каждая строка представляет собой название одного жанра.

## occupation.txt
Назначение: этот файл содержит список всех возможных профессий (родов занятий) пользователей системы.
Формат: простой текстовый файл, где каждая строка представляет собой название одной профессии.

## users.txt
Файл представляет собой текстовый файл с данными о пользователях, где каждая строка соответствует одной записи. 
Данные организованы в структурированном формате с использованием разделителей.
Структура данных:
Каждая строка содержит 6 полей, разделенных символом |.
Поля в каждой строке (в порядке следования):
ID: уникальный числовой идентификатор пользователя (целое число).
ФИО: полное имя пользователя в формате "Имя Фамилия".
Email: адрес электронной почты.
Пол: значение male (мужской) или female (женский).
Дата рождения: дата в формате YYYY-MM-DD.
Профессия/роль: название профессии или роли. Некоторые значения могут быть none (не указано).
